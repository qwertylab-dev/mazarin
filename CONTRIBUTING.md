# Contributing

For bug reports and feature requests, please search in issues first (including the closed ones). If there are no duplicates, feel free to [submit an issue][issues] with an issue template.

## Bug Report

When submitting a bug report, please make sure your issue contains **enough** information for reproducing the problem, including the steps to reproduce, device info, and other relevant details.

## Feature Request

**Mazarin** is an ebook reader designed to give users easy access to Project Gutenberg's extensive collection of free ebooks. We focus on feature requests that fit our core mission of ebook accessibility and a great reading experience.

If you'd like to request a feature, open a new Github issue with the Feature-Request template [here][feature-req].

## Pull Request

If you wish to contribute, please first leave a comment under the relevant issue or file a new issue describing the changes you are about to make.

We adhere to [conventional commits][conv-commits]. PRs should align with this convention.

## Building From Source

Fork this project, import and compile it with the latest version of [Android Studio](https://developer.android.com/studio/).
